# 🍱 Compress Images

image optimizer & WebP converter

## 👨🏻‍💻 Usage

with Node.js v12+

```bath

yarn install

```

Put your images in src folder.

### Optimize images

```bath

yarn run image:optimize

```

### Convert WebP images

#### with Node.js

```bath

yarn run image:webp

```

#### in bath

```bath

# mac
brew install webp

# win
choco install webp

```

```bath

yarn run image:webp-bath

```
